HTML

HTML stands for Hypertext Markup Language. HTML documents are what you see on most on your browser. They can have text content which describes ""what you see" and markup which describes "how it looks". This Video is a good overview of this.

Tags and Elements

HTML documents contain HTML elements and we tell the browser which element to use by using tags. We use tags to make things bold or to add italics We can also use tags to insert breaks or to format the page on how we want it to look. HTML 5.0 is the language that the web is programmed in today.

Why Computers are Stupid

Computers are stupid because they are only programmed to do what humans tell them to do. This also makes them hard to work with because you have to be very careful about your syntax and go back and debug if you want your software to work correctly.

Inline VS Block Elements

HTML Elements can be Inline or block. Block elements form an invisible box around the data inside of them. Inline does not.
